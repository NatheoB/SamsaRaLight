# Print a `sl_stand` object

Prints a compact, human-readable, one-line description of a `sl_stand`
object, summarizing the stand size, number of trees, number of sensors,
and grid geometry.

## Usage

``` r
# S3 method for class 'sl_stand'
print(x, ...)
```

## Arguments

- x:

  An object of class `sl_stand`.

- ...:

  Unused. Included for S3 method compatibility.

## Value

The input object `x`, invisibly.

## Details

This method is automatically called when a `sl_stand` object is printed
in the console.
